# 03. Convert ECMWF IFS HRES to WPS Input

## Objective

Prepare ECMWF IFS High Resolution (IFS HRES) GRIB1 data for the WRF Preprocessing System (WPS). This module demonstrates how to organize, separate, and crop the original ECMWF files into WRF-ready pressure-level and surface GRIB files.

---

## ECMWF Forecast Data

ECMWF provides several forecast datasets that can be used as input for numerical weather prediction.

### ECMWF Open Data (Free)

ECMWF offers an **Open Data** service that provides freely accessible forecast products with a reduced set of variables and resolutions suitable for education, research, and operational experimentation.

Information:

https://www.ecmwf.int/en/forecasts/datasets/open-data

---

### ECMWF IFS HRES (Subscription)

This training uses the **ECMWF Integrated Forecasting System (IFS) High Resolution (HRES)** dataset available through a subscription service.

Compared with the Open Data products, the subscribed IFS HRES dataset provides:

- Higher spatial resolution
- More atmospheric variables
- Complete pressure-level and surface fields
- Operational-quality forecast products
- Suitable for high-resolution WRF initialization

Information:

https://www.ecmwf.int/en/forecasts/datasets/set-i

---

## Workflow

The conversion process consists of the following steps.

```text
ECMWF IFS HRES (GRIB1)

            │

```

## Script Algorithm

The **convert_ifs_local.py** script automatically prepares ECMWF IFS HRES files before they are processed by WPS.

Its main workflow is illustrated below.

```text
Search Source Directory
          │
          ▼
Read All ECMWF GRIB Files
          │
          ▼
Skip Previously Processed Files
          │
          ▼
Copy Original GRIB
          │
          ▼
Separate Variables
   ├── Pressure-Level Fields
   └── Surface Fields
          │
          ▼
Crop to Indonesia Domain
          │
          ▼
Remove Temporary Files
          │
          ▼
Save Processing Log
          │
          ▼
Ready for WPS
```

---

## Algorithm Description

The script performs the following operations automatically.

1. Reads all ECMWF GRIB files from the source directory.

2. Prevents multiple instances from running simultaneously by creating a lock file.

3. Detects forecast initialization time and valid time directly from the ECMWF filename.

4. Organizes the output into directories based on the initialization cycle.

5. Copies the original GRIB file into the working directory.

6. Separates atmospheric variables into two independent datasets:

   - Pressure-level variables
   - Surface variables

7. Crops both datasets to the Indonesia model domain using **CDO**.

8. Removes temporary files to reduce disk usage.

9. Records every processed file inside a log file to prevent duplicate processing.

10. Produces WPS-ready GRIB files for the **ungrib.exe** step.

## Source Code

```python
#!/usr/bin/env python3
# -*- coding: utf-8 -*-

import os
import re
import shutil
import subprocess

# =====================================================
# SOURCE DIRECTORY
# =====================================================

SOURCE_DIR = (
    "/scratch/dmu/training_data/ecmwf_ifs/2026070600"
)

# =====================================================
# OUTPUT DIRECTORY
# =====================================================

BASE_OUTDIR = (
    "/scratch/dmu/training_data/ecmwf_ifs/2026070600/converted"
)

os.makedirs(BASE_OUTDIR, exist_ok=True)

# =====================================================
# LOG FILE
# =====================================================

log_file = os.path.join(
    BASE_OUTDIR,
    "processed_files.log"
)

processed_files = set()

if os.path.exists(log_file):
    with open(log_file, "r") as f:
        processed_files = set(
            line.strip() for line in f
        )

# =====================================================
# LOCK FILE
# =====================================================

lock_file = "/tmp/ifs_convert.lock"

if os.path.exists(lock_file):
    print("Another process is still running.")
    exit()

open(lock_file, "w").close()

# =====================================================
# DOMAIN INDONESIA
# =====================================================

lon_min = 89.5
lon_max = 145.5
lat_min = -15.5
lat_max = 15.5

# =====================================================
# EXECUTABLE
# =====================================================

grib_copy_exe = "grib_copy"
cdo_exe = "cdo"

try:

    # =================================================
    # LIST SOURCE FILES
    # =================================================

    source_files = sorted(
        [
            f for f in os.listdir(SOURCE_DIR)
            if f.endswith(".grib")
        ]
    )

    print(f"Total GRIB files: {len(source_files)}")

    # =================================================
    # LOOP FILES
    # =================================================

    for filename in source_files:

        # =============================================
        # FILTER ECMWF HRES
        # =============================================

        if not filename.startswith(
            "A1D"
        ):
            continue

        if "A1D" not in filename:
            continue

        # =============================================
        # SKIP IF ALREADY PROCESSED
        # =============================================

        if filename in processed_files:
            print(f"SKIP: {filename}")
            continue

        print("\n===================================")
        print("PROCESSING:")
        print(filename)

        # =============================================
        # PARSE INITIAL AND VALID TIME
        # =============================================

        match = re.search(
            r'R(\d{14})_(\d{14})',
            filename
        )

        if not match:
            print("Cannot parse time information")
            continue

        init_full = match.group(1)
        valid_full = match.group(2)

        init_folder = init_full[:10]
        valid_time = valid_full

        # =============================================
        # DIRECTORY
        # =============================================

        init_dir = os.path.join(
            BASE_OUTDIR,
            init_folder
        )

        converted_dir = os.path.join(
            init_dir,
            "converted"
        )

        os.makedirs(
            init_dir,
            exist_ok=True
        )

        os.makedirs(
            converted_dir,
            exist_ok=True
        )

        # =============================================
        # SOURCE FILE
        # =============================================

        source_grib = os.path.join(
            SOURCE_DIR,
            filename
        )

        # =============================================
        # LOCAL ORIGINAL FILE
        # =============================================

        local_grib = os.path.join(
            init_dir,
            f"ecmwf_A1_{valid_time}.grib"
        )

        print("Copying GRIB file...")

        shutil.copy2(
            source_grib,
            local_grib
        )

        print("Copied:")
        print(local_grib)

        # =============================================
        # TEMP FILE
        # =============================================

        tmp_pl = os.path.join(
            converted_dir,
            f"tmp_{valid_time}.pl.grib"
        )

        tmp_sfc = os.path.join(
            converted_dir,
            f"tmp_{valid_time}.sfc.grib"
        )

        # =============================================
        # FINAL FILE
        # =============================================

        final_pl = os.path.join(
            converted_dir,
            f"ecmwf_A1_{valid_time}_pl.grib"
        )

        final_sfc = os.path.join(
            converted_dir,
            f"ecmwf_A1_{valid_time}_sfc.grib"
        )

        # =============================================
        # SPLIT PL
        # =============================================

        cmd_pl = [
            grib_copy_exe,
            "-w",
            "editionNumber=1,typeOfLevel=isobaricInhPa",
            local_grib,
            tmp_pl
        ]

        print("Creating PL...")

        subprocess.run(
            cmd_pl,
            check=True
        )

        if not os.path.exists(tmp_pl):
            print("PL file not created")
            continue

        # =============================================
        # SPLIT SFC
        # =============================================

        cmd_sfc = [
            grib_copy_exe,
            "-w",
            "editionNumber=1,typeOfLevel=surface/depthBelowLandLayer/meanSea/heightAboveGround",
            local_grib,
            tmp_sfc
        ]

        print("Creating SFC...")

        subprocess.run(
            cmd_sfc,
            check=True
        )

        if not os.path.exists(tmp_sfc):
            print("SFC file not created")
            continue

        # =============================================
        # CROP PL
        # =============================================

        cmd_crop_pl = [
            cdo_exe,
            f"sellonlatbox,{lon_min},{lon_max},{lat_min},{lat_max}",
            tmp_pl,
            final_pl
        ]

        print("Cropping PL Indonesia domain...")

        subprocess.run(
            cmd_crop_pl,
            check=True
        )

        # =============================================
        # REMOVE TMP PL
        # =============================================

        if os.path.exists(tmp_pl):
            os.remove(tmp_pl)

        # =============================================
        # CROP SFC
        # =============================================

        cmd_crop_sfc = [
            cdo_exe,
            f"sellonlatbox,{lon_min},{lon_max},{lat_min},{lat_max}",
            tmp_sfc,
            final_sfc
        ]

        print("Cropping SFC Indonesia domain...")

        subprocess.run(
            cmd_crop_sfc,
            check=True
        )

        # =============================================
        # REMOVE TMP SFC
        # =============================================

        if os.path.exists(tmp_sfc):
            os.remove(tmp_sfc)

        # =============================================
        # SAVE LOG
        # =============================================

        with open(log_file, "a") as f:
            f.write(filename + "\n")

        print("\nDONE:")
        print(final_pl)
        print(final_sfc)

    print("\n===================================")
    print("ALL FILES COMPLETED")
    print("===================================")

finally:

    if os.path.exists(lock_file):
        os.remove(lock_file)
```
